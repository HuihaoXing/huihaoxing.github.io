Project Archive

Boston Bus Equity Analysis

Led metric design and 5-year EDA for GBH News (NPR affiliate). Reconstructed a passenger-centric delay metric using next-available-bus logic—revealing systematic underreporting hidden by standard KPIs.

Route 14
>16 min delay
5 Years
Data Analyzed
Metric DesignEDATime SeriesPython/SQL
▸ DEEP DIVE

◆ Why This Problem

Existing MBTA KPIs masked real passenger experience: a bus leaving 2 min early is counted 'on time' but strands waiting passengers.

◆ How It Was Built

Constructed available_bus_depart_time—earliest bus actually catchable. Compared 2019 vs 2022 ridership. Joined MBTA data with 2023 Passenger Survey demographics.

◆ Key Finding

High-delay routes (14, 23, 28, 111) disproportionately serve lower-income communities. Route 14 averaged >16 min. Routes 23/28 improved ~3 min post-COVID; Route 24 worsened by 7 min.

◆ Limitation

Fall seasons only. AFC undercounting may bias boardings data.

MBTA Delay Prediction (Azure)

End-to-end delay prediction for Green Line B integrating weather and event data. Cloud ETL on Azure; XGBoost by AutoML. Weather and events confirmed as drivers via ANOVA and T-test.

◆ Why This Problem

Fenway and TD Garden events create predictable surges—forecasting from event schedules + weather lets MBTA pre-position resources.

◆ How It Was Built

Merged MBTA arrivals with OpenWeather and Ticketmaster data. ETL via Spark + Azure Data Factory. AutoML tested 20+ configs; XGBoost won.

R²=0.34—unobserved variance from mechanical/signal failures. Live ridership would improve significantly.

Project Archive

Title A/B Experiment (Tencent)

Rigorous A/B test: LLM-optimized vs. DeepL-translated titles for a 400k DAU tech channel. Power analysis: baseline CTR=12%, MDE=3%, α=0.05 → 128k samples/group.

+4pp
CTR Lift
128k
Samples/Group
A/B TestingPower AnalysisHypothesis TestingLLM Prompt
▸ DEEP DIVE

◆ Why This Problem

DeepL titles are literal—underperform in Chinese social media CTR. Hypothesis: LLM titles with localized framing improve CTR without hurting completion (guardrail).

◆ How It Was Built

User-level randomization to prevent contamination. Fixed prompt template for LLM; DeepL baseline for control. Sample size via two-proportion Z-test.

◆ Key Finding

Simulated CTR: 12%→16% (+4pp, p<0.05). Completion rate delta not significant—quality intact. Supports LLM titles as default.

◆ Limitation

Results are simulated. Novelty effects could inflate short-term CTR. Recommend 2-week holdout.

◆ Why This Problem

Single-model approaches plateau on tabular churn. Stacking captures different bias-variance trade-offs across tree algorithms.

◆ How It Was Built

10-fold outer + 5-fold inner CV for target encoding. Three models combined via rank-averaging + scipy-optimized weights. N-gram features captured interaction effects.

Rank-averaging beat simple averaging. Charges-deviation (TotalCharges − tenure × MonthlyCharges) was top churn driver.
